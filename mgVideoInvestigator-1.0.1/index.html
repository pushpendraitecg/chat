Magnoliyan Video Chat Investigator

This is simple tool used to investigate if you will be able to deploy full featured Video Chat server on a hosting platform.

Make sure to update source/server/bin/config.php if you want to change server port.

Then, run your server from command line or try web commander

Then open this file index.html and also update domain and port in mgVideoChat javascript initialization and reload the page from a browser to see result.
